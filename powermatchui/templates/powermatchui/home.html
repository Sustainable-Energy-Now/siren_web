Welcome to Powermatch Web!

Siren Web is a web interface to Power Match, an open source python app for matching and balancing Renewable Energy resources to the load on the South West Interconnected system(SWIS). Power Match takes as input load and generation data from Siren. Powermatch quantifies and costs the dispatchable energy generation, storage and CO2-e emissions. The parameters passed to Power Match are stored in database tables that can bemaintained using the Table Update function above. In this main panel you can select the load year from the available data shown in the drop down list below. Select the level of detail and click submit to run Power Match.
