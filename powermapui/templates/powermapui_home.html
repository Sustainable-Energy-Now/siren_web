This is a placeholder for a web interface to the Siren Powermap module. The purpose of this interface is to explain the data used by Powermap to model renewable energy generation in the South West Interconnected System (SWIS). Currently to run the actual Powermap requires installing the python application and setting up the inputfiles required. Power Map takes as input weather and facilities data. Powermap uses th System Advior Model (SAM) developed by the National Renewable Enegery Laboratory (NREL). To use Powermap begin by creating a Scenario. This is a collection of Facilities of different technologies. Output from Powermap is used as input to he Powermatch module. Currently output for the demand year of 2022 has been loaded into the database.
