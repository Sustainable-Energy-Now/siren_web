This is a web interface to the Siren Powermap module. Powermap is used to model renewable energy generation in the South West Interconnected System (SWIS). Power Map takes as input weather and facilities data. Powermap uses th System Advior Model (SAM) developed by the National Renewable Enegery Laboratory (NREL). To use Powermap begin by creating a Scenario. This is a collection of Facilities of different technologies.
